- **Frontend:** React + Vite

- **Language:** TypeScript
- **Tooling:** ESLint, Prettier
- **Deployment:** *(planned: AWS)*

This stack was chosen for:
- fast development with Vite
- utility-first styling with Tailwind

- scalability for future full-stack features
